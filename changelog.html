1.12.0 July 4, 2025

  • Now requires Openfire 5.0.0 or later
  • [#7] - Add endpoints that can get, set and delete user vCards.

  • [#203] - Reduce log level verbosity for some errors.

  • Now requires Openfire 4.8.0 or later

  • [#195] - Fix API incompatibilities with Openfire 4.8.0
